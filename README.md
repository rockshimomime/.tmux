# Tmux


## Configurações

### Baixar o TPM

```bash
git clone https://github.com/tmux-plugins/tpm ~/.tmux/plugins/tpm
```

### Instalar os plugins

Na janela do Tmux rodar o comando 
```bash
 prefix + I
 ```
 
 
 ### Criar o link simbólico
 
 Criar o link simbólico para mandar o `tmux.conf` para a `~`.
 
 ```bash
 ln -s  ~/.tmux/.tmux.conf ~/.tmux.conf  
 ```
